Project Summary

1. This is a collection of colab notebooks where I implemented deep learning models from scratch for Computer Vision and Natural Language Procssing tasks.

2. Used TensorFlow 2 and Keras APIs to build the models using datasets from Kaggle and custom datasets.

3. Presence of Cellphone Detector: In this notebook I implemented a computer vision model using Convolutional Neural Network that classifies images with or without mobile phones. Initially I only used this Kaggle dataset to train the model but the dataset wasn't good enough both in terms of number of images and variety. So I created a dataset myself to solve this problem, which helped me to understand the importance of data in developing a effective model. Also used Data Augmentation technique from Tensorflow to improve the model performance.

4. News Category - Detector: In this notebook I built a Natural Language Processing model to classify news by analyzing the sentiment of the news headlines. Through this project I understood the data proceesing techniques for NLP tasks before feeding the data to the model which includes tokenization,encoding,embeddingetc. I also got familier with different types of hidden layers used in the neural network for NLP tasks such as Recurrent Neural Netowrk(RNN) and Long Short-Term Memory(LSTM).

5. Real/Fake Face - Detector: In this notebook I developed a deep convolutional neural network that can successfully recognize a real image and a fake/photoshpped image of a human face. I used this kaggle dataset to train the implemented model. I used a subset a data from dataset for training purpose. Apart from building an effective computer vision model, I also understood things like data normalizing, early stopping etc.
