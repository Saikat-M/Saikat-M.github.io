Project Summary

This project placed 5th place in the AI Challenge organized by Viam Robotics: Viam Post

1. It's a portable personal assistant device built using raspberry pi 4. It leveraging Large Language Model through Google's Gemini API and also use Viam SDK for person recognition.

2. It is capable of answering questions based on voice input and the answers can be heard via speaker. It can also take picture input and answer questions regarding it.

3. The conversation is triggered after it detects a person using the Pi Camera and the detection is done using model named 'EfficientDet-COCO' from Viam.

4. It is capable of continuing the conversation through follow-up questions.
