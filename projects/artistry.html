Artistry: An Image Generation & Enhancement App

  • Demo: LinkedIn Post
  • Github URL: Project Link
  • Tech Stack & APIs: React.js, Stability.ai APIs, ChatGPT-3.5-Turbo

Project Summary

1. A React based web app to generate & refine artwork with prompt assistance.

2. User can use this app to create new images which can be further used to make a story line consisting of individual images i.e. an album.

3. This app can also enhance an existing image(like drawing or sketches) using text prompts.

4. The app can be used to get prompt suggestions. Once the user explains how the picture should look like or what is the scene about in a particular story line, it can generate prompt which can be used to perform the task.

4. The app leverages Stability.ai APIs for image generation and enhancement tasks and OpenAI's GPT-3.5-Turbo API for prompt suggestion.

5. The target audience of this application is anybody who's interested in arts. Entertainment industry can be one domain but it's use cases are not limited to that space only.
